# AIコーラー — 完全AI架電サービス LP

株式会社アイドマ・ホールディングスが提供する、AIで完全自動架電する「AIコーラー」のサービス紹介LPです。

## 主要訴求ポイント

- **初期設定費 30万円** (全プラン共通、オンボーディング込み)
- **コール単価 25〜30円** (1,000コール単位で購入、ボリュームディスカウントあり)
- **システム利用料 無料** (月額固定費ゼロ)
- **全国法人データ 約210万社** をプラン関係なく無料利用可能
- **好きなタイミングで好きな数の架電が可能** (オンデマンド方式)
- **マネジメント工数ゼロ** — シフト・教育・離職対応すべて不要
- 松竹梅 3プラン構成 + 無料トライアル100コール

## セクション構成

1. Hero — メインキャッチ・CTA
2. Services — 3つの強み(AI自動架電 / 法人DB / スクリプト最適化)
3. ClientLogos — ご利用業界例
4. Results — 数字で見る強み(コール単価/マネジメントZERO/210万社/利用料0円)
5. Pricing — 松竹梅プラン + 共通費サマリーバー
6. Step — 導入の流れ(4ステップ)
7. FAQ — よくある質問
8. Contact — 無料トライアル100コール 申込フォーム

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 で起動します。

## ビルド・本番

```bash
npm run build
npm run start
```

## 環境変数

問い合わせフォームのメール送信に [Resend](https://resend.com/) を利用しています。

```env
RESEND_API_KEY=your_resend_api_key
```

送信先メールアドレスは `src/app/api/contact/route.ts` で指定しています。
