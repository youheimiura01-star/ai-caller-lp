# 音声サンプル配置先

本番のAI架電音声サンプルファイルをこちらに配置してください。

## 想定ファイル

| ファイル名 | 内容 | 推奨形式 |
|---|---|---|
| `sample.mp3` | Hero表示用のAI架電サンプル(15〜30秒推奨) | MP3 / 128kbps以上 |

## 配置後の動作

`public/audio/sample.mp3` を配置するだけで、Heroのミニプレイヤーが自動で再生可能になります(コード変更不要)。

ファイルが存在しない場合は、UIに「サンプル音声準備中」と表示されます。

## 推奨内容(参考)

- 受付突破 → 担当者接続 → 簡単な商品説明 → アポ取得までを15〜30秒で
- 冒頭にナレーション「以下はAIによる実際の架電サンプルです」を入れると安心感UP
